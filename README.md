# Code Review Dashboard

A Flask-based web application for submitting and tracking code review issues.

## Features
- Submit code snippets for review
- Log issues with severity levels (Low, Medium, High)
- Suggest fixes for each issue
- View all submitted reviews on a dashboard

## Tech Stack
- Python
- Flask
- HTML/CSS
